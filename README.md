# Discord Bot - Moderation & Music

Feature-rich Discord bot with auto-moderation, music playback, XP leveling, and slash commands.

## Features
- 🛡️ Auto-moderation (ban, kick, warn, timeout, anti-spam)
- 🎵 Music playback from YouTube (queue, skip, pause)
- 📊 XP leveling system with leaderboard
- 👋 Welcome messages + auto-roles
- 🎫 Ticket system
- 🎁 Giveaways

## Tech Stack
- Discord.js v14, @discordjs/voice, ytdl-core, MongoDB, TypeScript

## Getting Started
```bash
npm install && cp .env.example .env && npm run dev
```
